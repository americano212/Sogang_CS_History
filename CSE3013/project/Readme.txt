visual studio 용 cpp openframeworks를 다운받는다
폴더 내 ../script/templates/vs 내부에 
bin/data와 src 디렉토리 안에 
압축 해제한 data와 src파일을 넣는다.

sln파일로 실행하고 ctrl+F5로 실행시킨다.

초기화면에서 [S]를 누르면 시작할 수 있다.
WASD로 조작하며 pac-man을 조정할 수 있다.

체리를 먹으면 스테이지를 클리어 할 수 있고
[N]으로 Next 스테이지에 갈 수 있다.
게임을 종료하고 싶을 땐 [R]을 눌려서 점수를
저장하고, 메인메뉴로 돌아와 [Q]를 눌려 종료한다.
